![맛집 지도 홈페이지1](https://user-images.githubusercontent.com/64799496/133057534-ceed6673-83f4-411f-91cb-3d2ca66f5a60.PNG)

**맛집 지도**
- 소개 : 서울 맛집들을 지도에 표시한 홈페이지
- 기능 
   * 맛집 정보 스크래핑
   * 지도 보여주기
   * 각 맛집 별 마커,정보창,카드 만들고 서로 연결하기
- 기술
   * Selenium으로 'SBS TV 맛집 사이트'에서 맛집정보 스크래핑
   * 네이버 지도API 이용
   * 맛집 정보 DB에 저장
